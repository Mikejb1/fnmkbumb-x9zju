Datenschutz

  • Portfolio-Daten clientseitig mit AES-GCM (PBKDF2 150k Iterationen) verschlüsselt — Worker speichert nur unentschlüsselbaren Blob in Cloudflare KV
  • Master-Code wird niemals übertragen — Entschlüsselung passiert nur im Browser
  • KI-Anfragen senden aggregierten Portfolio-Kontext an Anthropic API (über Cloudflare Worker)
  • Chat-Gedächtnis wird als Klartext-Zusammenfassung in deinem KV-Slot gespeichert — nur du mit deinem Master-Code kannst es entschlüsseln
  • Krypto-Preise via CoinGecko (öffentlich, keine Auth), Edelmetalle via Worker-Proxy
  • Lokales API-Usage-Tracking nur in localStorage — keine Server-Speicherung
Backup & Export
